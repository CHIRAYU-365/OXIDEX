## 📑 Open-Source Amendments for Decentralized Ecosystems

Because this project is built for Web3 blockchains and includes smart contract components, the following amendments apply:

### 1. Fork Safety and Naming
You are welcome to fork this project and deploy your own matrices. However, you **must not** use the name "OXIDEX" for your main application or contract name if you make structural changes to the logic. This is to prevent confusion and protect users from modified smart contracts using the original brand name.

### 2. Smart Contract Deployment Responsibility
Deployment of the smart contracts to any live blockchain network is done entirely at your own risk. The authors of this codebase have no control over, nor access to, any funds or fees sent to contracts deployed by third parties.

*   User wallet security or compromised private keys.
